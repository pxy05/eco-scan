DOWNLOAD INSTRUCTIONS:
======================

1. Try the direct links provided above first
2. If links don't work, you can export from Figma:
   - Open your Figma file
   - Select each asset layer
   - Export as PNG at the appropriate resolution
   - Save with the hash filename (e.g., afde586264834bcd046545c0d95a8cf848cc7628.png)

3. For Cursor migration, replace "figma:asset/" imports with relative paths:
   Example: import imgEarthCute from "./assets/afde586264834bcd046545c0d95a8cf848cc7628.png";
